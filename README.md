# Procurement-Planning-Program
Заметки — это простое и удобное мобильное приложение для создания, хранения и поиска заметок с параметрами. Работает прямо в браузере телефона без установки и интернета. Все данные хранятся локально на устройстве, обеспечивая конфиденциальность и доступность в любое время.
