Заметки: Мобильное приложение для хранения и поиска заметок с параметрами

Заметки — это простое и удобное мобильное приложение для создания, хранения и поиска заметок с параметрами. Работает прямо в браузере телефона без установки и интернета. Все данные хранятся локально на устройстве, обеспечивая конфиденциальность и доступность в любое время.



📌 Основные возможности





Создание и редактирование заметок





Быстрое создание заметок с помощью кнопки + Новая заметка.



Автосохранение через 0.6 секунды после остановки ввода.



Подтверждение сохранения с помощью статуса √ Сохранено.



Удаление заметок с подтверждением.



Система параметров





Каждая заметка может иметь произвольный набор параметров.



Формат параметров: #Категория Значение (например, #Организация Водоканал Тихонов).



Поддержка иерархии параметров через пробелы.



Автодополнение при вводе параметров





Подсказки из существующих параметров при вводе.



Навигация по подсказкам с помощью стрелок, выбор — Enter или Tab.



Учёт иерархии и частичного ввода.



Фильтрация заметок





Фильтр открывается кнопкой □ в шапке списка.



Поиск по одному или нескольким токенам.



Отображение счётчика: "5 из 23 заметок".



Возможность сохранять фильтры и быстро переключаться между ними.



Просмотр всех параметров





Экран "Все параметры" — иерархический список всех параметров.



Нажатие на параметр фильтрует список заметок.



Хлебные крошки для навигации по уровням иерархии.



Удаление параметра сразу из всех заметок.



Печать





Кнопка □ формирует страницу для печати.



Печатаются только заметки из текущего фильтра.



Чистый формат без лишних элементов интерфейса.

🚀 Как запустить





Локально:





Откройте файл index.html в браузере телефона.



Приложение работает без интернета и не требует установки.



С использованием локального сервера (для разработки):





Установите зависимости: pip install -r requirements.txt.



Запустите сервер: python server.py.



Откройте http://localhost:8000 в браузере.

📄 Пример оформления параметров

Параметры пишутся в отдельное поле — каждый с новой строки. Первый токен после # — категория, остальные — уточнения:

#Организация Водоканал Тихонов
#Тип Важный
#Статус В работе
#Дата 2026 Март
